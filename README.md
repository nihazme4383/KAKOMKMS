# KAKOM KMS

Tapak statik awal untuk KAKOM Kolej Matrikulasi Selangor.

## Kandungan

- `index.html` - struktur laman utama
- `styles.css` - reka bentuk responsif
- `app.js` - kiraan masa, senarai acara, jadual draf dan rekod keputusan sementara
- `assets/kakom-mark.svg` - tanda KAKOM sementara berdasarkan keterangan logo rasmi

## Cara guna

Buka `index.html` terus dalam pelayar. Keputusan yang direkod melalui borang disimpan dalam `localStorage` pelayar komputer semasa.

Untuk produksi sebenar, pindahkan rekod keputusan kepada backend atau pangkalan data supaya data boleh dikongsi oleh semua pengguna.
